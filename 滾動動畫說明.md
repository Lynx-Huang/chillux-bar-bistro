## 🎮 動畫控制原理

### 滾動距離映射
```javascript
// 計算元素在視窗中的進度 (0-1)
const progress = (viewportBottom - elementTop) / totalAnimationHeight;

// 應用不同類型的變換
switch (animationType) {
    case 0: // 從左滑入
        translateX = (1 - progress) * -100;
    case 1: // 從右滑入  
        translateX = (1 - progress) * 100;
    case 2: // 縮放效果
        scale = 0.8 + (progress * 0.2);
    case 3: // 從下滑入
        translateY = (1 - progress) * 50;
}
```

### 性能最佳化
- ✅ **防抖處理**：使用 `requestAnimationFrame` 限制動畫頻率
- ✅ **單次觀察**：動畫完成後停止觀察元素
- ✅ **硬體加速**：使用 `transform` 和 `opacity` 觸發GPU加速
- ✅ **條件執行**：只在元素可見時執行動畫

## 🛠️ 自訂動畫

### 添加新的滾動動畫元素
1. **HTML中添加類別**：
```html
<div class="fade-on-scroll scroll-delay-2">您的內容</div>
```

2. **可用的動畫類別**：
- `.fade-on-scroll` - 基礎淡入動畫
- `.scroll-delay-1` 到 `.scroll-delay-5` - 延遲動畫
- `.scroll-scale` - 縮放動畫
- `.scroll-slide-left` - 左滑動畫
- `.scroll-slide-right` - 右滑動畫

### 調整動畫參數
在 `mystyles.css` 中修改：
```css
/* 動畫持續時間 */
.fade-on-scroll {
    transition: all 0.8s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 動畫延遲 */
.scroll-delay-1 { transition-delay: 0.1s; }
```

### 新增計數器
```html
<div class="counter" data-target="100">0</div>
<div class="counter-label">您的標籤</div>
```

# 劉姿妤的個人履歷網站

這是一個精美的響應式履歷網站，基於 Bootstrap 5 框架開發，具有以下特色：

## ✨ 主要特色

### 🎨 視覺設計
- **優雅配色**：採用玫瑰金色系（#B76E79）搭配金色強調色（#DAA520）
- **現代字體**：使用 Google Fonts 的思源黑體（Noto Sans TC）
- **漸層背景**：多層次漸層背景搭配微妙紋理效果
- **玻璃擬態**：毛玻璃效果的卡片設計

### 🎭 動畫效果
- **頁面載入動畫**：平滑的進入效果
- **打字機效果**：主標題的打字動畫
- **滾動進度條**：頂部顯示瀏覽進度
- **技能進度條**：動態技能熟練度展示
- **懸停效果**：互動式按鈕和圖示
- **滾動觸發動畫**：區塊滾動進入時的動畫

### 📱 響應式設計
- **行動裝置優化**：完美適配手機和平板
- **流暢導覽**：固定式側邊欄導覽
- **觸控友善**：針對觸控設備優化的按鈕大小

### 🔧 技術特色
- **CSS 變數**：易於自訂主題顏色
- **模組化 CSS**：結構清晰的樣式檔案
- **JavaScript 增強**：平滑滾動和互動效果
- **SEO 友善**：語義化 HTML 標籤

## 🚀 快速開始

1. **準備照片**
   - 準備一張 500x500 像素的正方形大頭照
   - 將照片重新命名並放置到 `assets/img/` 資料夾中
   - 更新 `index.html` 中的圖片路徑

2. **編輯個人資訊**
   - 在 `index.html` 中替換所有個人資訊
   - 更新聯絡方式、社群媒體連結
   - 修改工作經歷、教育背景、技能等內容

3. **自訂樣式**
   - 在 `css/mystyles.css` 中調整顏色、字體等樣式
   - 修改 CSS 變數來快速更換主題色彩

## 🎨 色彩配置

```css
:root {
    --bs-primary-rgb: 183, 110, 121;  /* 主色：玫瑰金 */
    --bs-secondary-rgb: 240, 230, 220; /* 輔色：淡米色 */
    --accent-color: rgb(218, 165, 32);  /* 強調色：金色 */
}
```

## 🛠️ 自訂指南

### 更換主題顏色
在 `mystyles.css` 中修改 `:root` 變數：

```css
:root {
    --bs-primary-rgb: 您的RGB值;
    --accent-color: rgb(您的RGB值);
}
```

### 添加新區塊
1. 複製現有的 `<section>` 區塊
2. 修改 `id` 和內容
3. 在導覽列中添加對應連結

### 調整動畫效果
在 `mystyles.css` 中找到對應的 `@keyframes` 規則進行修改，或在 `scripts.js` 中調整 JavaScript 動畫。
